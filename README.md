=========================================
          MEMORY GAME — FLIP & MATCH
=========================================

This is a simple and fun Memory Card Game built using
HTML, CSS, and JavaScript. The goal is to flip the cards,
find matching pairs, and complete each level within the
assigned move limit.

-----------------------------------------
   FEATURES
-----------------------------------------
- Multiple levels (2x2 to 8x8 grid)
- Dynamic move limit based on grid size
- Timer to track how long the player takes
- Win popup shows:
     - Total Moves
     - Move Limit
     - Time Taken
     - Star Rating
- Loss popup shows:
     - Moves Used
     - Move Limit
     - Time Taken
- Try Again / Next Level options
- Emoji symbols for fun gameplay

-----------------------------------------
   MOVE LIMIT FORMULA
-----------------------------------------
Move Limit = GridSize × GridSize

Example:
2×2 → 4 moves
3×3 → 9 moves
4×4 → 16 moves
5×5 → 25 moves
6×6 → 36 moves
7×7 → 49 moves
8×8 → 64 moves

-----------------------------------------
   HOW TO PLAY
-----------------------------------------
1. Click a card to flip it.
2. Flip another card — if both match, they stay open.
3. Match all pairs to complete the level.
4. Finish within the Move Limit to win.
5. If moves reach the limit → Game Over popup appears.
